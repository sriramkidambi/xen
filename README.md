# Xen

An opinionated fork of [Bridle](https://github.com/neiii/bridle) by [d0](https://github.com/neiii) - tailored for my personal workflow.

This is my personal configuration manager for AI coding assistants. If you find it useful, great! But all credit and applause go to the original Bridle project.

## Credits

**This project is based entirely on [Bridle](https://github.com/neiii/bridle) by d0 ([@neiii](https://github.com/neiii)).**

Bridle is the original unified configuration manager for AI coding assistants. This fork exists purely for my own opinionated customizations. If you're looking for the real deal, please use the original:

- **Original Project:** [github.com/neiii/bridle](https://github.com/neiii/bridle)
- **Install Bridle:** `npx bridle-ai` or `npm install -g bridle-ai`

## What is this?

Xen is my personal take on Bridle. It manages profiles, installs skills/agents/commands, and switches configurations across Claude Code, OpenCode, Goose, Amp, Copilot CLI, and Crush.
